## 🎯 About the Project

**Donatikov.net** is a desktop application designed to simulate a digital wallet ecosystem. It provides a user-friendly interface to manage personal funds and pay for various digital services, such as games, software subscriptions, and mobile balances.

### Key Features:
* **User Accounts:** Secure login and profile management.
* **Wallet System:** Real-time balance tracking and transaction history.
* **Service Integration:** A catalog of popular services available for top-ups.
* **Modern UI:** Built with a focus on intuitive user experience.
